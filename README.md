# 🛍️ SmartMall - ศูนย์การค้าออนไลน์

เว็บไซต์ขายของออนไลน์ที่สมบูรณ์สร้างด้วย Flask พร้อมฟีเจอร์ครบครัน

## ✨ ฟีเจอร์หลัก

- ✅ **หน้าแรก** - แสดงสินค้า 6 รายการพร้อมรูปภาพสวยงาม
- ✅ **ระบบตะกร้า** - เพิ่ม/ลบสินค้า อัปเดตจำนวน และแสดงจำนวนในตะกร้า
- ✅ **ตัวกรอง** - กรองสินค้าตามหมวดหมู่ (อิเล็กทรอนิกส์, คอมพิวเตอร์, กล้อง)
- ✅ **หน้าชำระเงิน** - ฟอร์มกรอกข้อมูลลูกค้า วิธีการชำระเงิน (บัตร/โอนเงิน/อีวอลเล็ต/เก็บเงินปลายทาง)
- ✅ **รูปภาพสินค้า** - สร้างอัตโนมัติ 6 รูปสินค้าตัวอย่างสวยงาม
- ✅ **ออกแบบสวย** - UI/UX ดีงาม Responsive ทั้งมือถือและเดสก์ทอป

## 📦 ติดตั้ง

### ข้อกำหนด
- Python 3.8+
- pip

### ขั้นตอนการติดตั้ง

```bash
# 1. สร้าง Virtual Environment
python -m venv venv

# 2. เปิดใช้ Virtual Environment
# บน Windows:
venv\Scripts\activate
# บน macOS/Linux:
source venv/bin/activate

# 3. ติดตั้ง Dependencies
pip install -r requirements.txt

# 4. รันแอปพลิเคชัน
python app.py
```

## 🚀 การใช้งาน

1. เปิดเบราว์เซอร์และไปที่ `http://localhost:5000`
2. ท่องดูสินค้า - คลิกกรองตามหมวดหมู่
3. เพิ่มสินค้าลงตะกร้า
4. ไปที่หน้าตะกร้าเพื่อดูสรุปสินค้า
5. คลิกชำระเงินและกรอกข้อมูลของคุณ
6. เลือกวิธีการชำระเงิน
7. ยืนยันการสั่งซื้อ

## 📂 โครงสร้างโปรเจค

```
SSS/
├── app.py                 # Flask application หลัก
├── requirements.txt       # Python dependencies
├── templates/             # HTML templates
│   ├── index.html        # หน้าแรก
│   ├── cart.html         # หน้าตะกร้า
│   ├── checkout.html     # หน้าชำระเงิน
│   └── success.html      # หน้าสำเร็จ
└── static/               # Static files
    ├── css/
    │   └── style.css     # สไตล์หลัก
    └── images/
        └── products/     # รูปภาพสินค้า (สร้างอัตโนมัติ)
```

## 🎨 เทคโนโลยี

- **Backend**: Flask 2.3.0
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Image Generation**: Pillow 9.5.0
- **Session Management**: Flask Sessions
